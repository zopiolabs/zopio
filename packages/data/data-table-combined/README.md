# @repo/data-table

Full-featured table management system for the Zopio framework.

## Included Features

- ✅ Pagination & total count
- ✅ Sorting
- ✅ Filtering (multi-field, typed)
- ✅ Column layout builder (visibility, order, width)
- ✅ Row selection & bulk delete
- ✅ CSV export
- ✅ Inline editing
- ✅ Search input
- ✅ Live updates (polling)
- ✅ URL sync with query params

# Zopio Onboarding Package

This package provides a lightweight wrapper around [Reactour](https://github.com/elrumordelaluz/reactour) to deliver in-app product tours and onboarding flows.

## Features
- Configurable guided steps
- Easy integration with any page
- Fully client-side compatible

## Usage
1. Wrap your layout with `<UserTourProvider>`.
2. Trigger tour with `useUserTour` hook.
3. Add `data-tour="step-X"` to elements.
